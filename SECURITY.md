# Security

## Reporting a vulnerability

Please report security issues via [GitHub Security Advisories](https://github.com/JeelVankhede/frontend-ai-starter-recipes/security/advisories/new) for this repository.

## Scope

This project is a **local CLI template generator**. It does not run a network service or collect telemetry by default. Generated files may reference third-party monitoring tools (e.g. Sentry) as **documentation only** — configuring them is up to the consuming project.
