# Contributing & support

## Feature requests

Open a **[GitHub Issue](https://github.com/JeelVankhede/frontend-ai-starter-recipes/issues)** with:

- **Problem** — friction today  
- **Proposal** — new preset, rule, skill, or adapter  
- **Example** — stack and sample output if possible  

## Bug reports

Include **Node version**, **package version**, full **command** run, and **expected vs actual** behavior.

## Contributing code

See **[CONTRIBUTING.md](https://github.com/JeelVankhede/frontend-ai-starter-recipes/blob/main/CONTRIBUTING.md)** in the repo: branch naming, tests, docs build.

## Security

**[SECURITY.md](https://github.com/JeelVankhede/frontend-ai-starter-recipes/blob/main/SECURITY.md)** — report via GitHub Security Advisories.
